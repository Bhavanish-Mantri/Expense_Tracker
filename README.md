# Expense Tracker 

A simple and beginner-friendly Expense Tracker web application built using HTML, CSS, and JavaScript.

This project allows users to add income and expense transactions and instantly view the updated balance, total income, and total expenses in a clean and minimal interface.

## Live Demo

https://bhavanish-mantri.github.io/Expense-tracker/

## Features

- Add transactions as Income or Expense  
- Toggle buttons to select transaction type  
- Automatic balance, income, and expense calculation  
- Transaction history display  
- Clean and responsive UI  
- Fully frontend-based (no backend)

## Technologies Used

- HTML  
- CSS (Flexbox and basic styling)  
- JavaScript (DOM manipulation and event handling)

## Project Structure

```
Expense-tracker/
```

## Usage

- Enter a transaction description  
- Enter the amount  
- Select Income or Expense  
- Click on **Add Transaction**  
- View updated balance and transaction history instantly

## Note

- Data is not stored permanently (page refresh resets data)
